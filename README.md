# Text Analysis and Classification using ML

## 📌 Project Overview
This project performs text analysis and sentiment classification using machine learning techniques.

## 🔹 Features
- Resume information extraction using regex
- Sentiment analysis on text data
- Comparison of multiple ML models

## 🛠 Tools Used
- Python
- scikit-learn
- nltk
- matplotlib

## 🤖 Models Used
- Naive Bayes
- SVM (LinearSVC)
- Random Forest

## 📊 Evaluation Metrics
- Accuracy
- Precision
- Recall
- F1 Score

## 📈 Output
- Model comparison graph
- Sentiment prediction results
