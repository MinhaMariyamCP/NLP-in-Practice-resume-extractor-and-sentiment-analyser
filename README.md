# Text Analysis and Classification using ML

## 📌 Project Overview
This project performs text analysis and sentiment classification using machine learning techniques. It includes a resume information extractor and a sentiment analysis system using multiple ML models.

## 🔹 Features
- Resume information extraction using regex (email & phone)
- Sentiment analysis on text data
- Comparison of multiple ML models

## 🛠 Tools Used
- Python
- scikit-learn
- nltk
- matplotlib

## 🤖 Models Used
- Naive Bayes
- SVM (LinearSVC)
- Random Forest

## 📊 Evaluation Metrics
- Accuracy
- Precision
- Recall
- F1 Score

## 📈 Output
- Model comparison graph
- Sentiment prediction results

   pip install pandas scikit-learn nltk matplotlib seaborn textblob

2. Run the notebook:
   sentiment_analysis.ipynb
